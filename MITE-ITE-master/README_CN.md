# MITE-ITE
一个为MITE-1.6.4 R196增加难度和加入新东西的模组
## 安装方式
1. 下载[FishModLoader](https://github.com/XiaoYuOvO/FishModLoader/releases)
2. 安装FishModLoader到你的MITE1.6.4 R196中
3. 下载MITE-ITEPlugin.jar
4. 将MITE-ITEPlugin.jar放入游戏文件夹中的mods文件夹中
5. 启动游戏
